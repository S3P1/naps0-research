Why this evaluation matters

01
Beyond clip-level accuracy
Standard accuracy evaluates pre-segmented clips in isolation. Real monitoring requires detecting events in a continuous stream — a harder problem that clip-level accuracy systematically overestimates. Naps0 is evaluated over 200 synthetic streaming episodes using PSDS, the DCASE sound event detection standard (Bilen et al., ICASSP 2020).
02
Threshold independence
Reporting a single F1 at a fixed threshold conceals how the system behaves across its operating range. PSDS is the area under the PSD-ROC curve — evaluated across 18 thresholds simultaneously, providing a threshold-independent score that enables genuine system comparison.
03
Label auditing
Non-clinical crowdsourced datasets contain mislabeled clips that inflate metrics. We apply an acoustic criterion combining F0 profile, spectral centroid, and cross-model agreement to challenge source labels. 91 pain clips were excluded after analysis — criterion documented and reproducible.

Four public sources,
one scientific standard

Training data is drawn from four publicly available datasets. Each clip is processed through a multi-criteria label auditing pipeline before inclusion — source labels from non-clinical collections are challenged, not accepted at face value.

The validation set is fixed, original clips only, never augmented, never modified.
