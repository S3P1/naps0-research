# Naps0

**Real-time, privacy-first baby cry monitor for Android.**

Naps0 listens continuously to a baby's cry and classifies the underlying need in real time — entirely on the device, with no cloud, no data upload, and no subscription.

## What it does

Classifies infant crying into four categories:

- **Pain** — immediate alert, highest priority
- **Hunger**
- **Discomfort**
- **Tired**

The entire inference pipeline runs on-device. No audio is ever transmitted to any server.

## Key properties

- On-device inference — no cloud dependency
- Privacy by design — audio never leaves the phone
- Real-time classification with sub-100ms onset latency
- Differentiated alerts per class
- Android, no subscription, no ads
